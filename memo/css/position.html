-POSITION-






positionではそれぞれの要素の位置を変えることができる

基本的にはcssで設定してhtmlにある要素の位置関係を変化させる

top.bottom.left.rightなどを用いてどこに移動させるかを決めることができる。


"<div class=box></div>"に対する処理

position:relative;

position:relativeは今の位置からの絶対的な位置を決める

現状の上から10px、左から20px動かす場合

    .box{
        position: relative;
    top: 10px;
    left: 20px;
}

横列の真ん中に動かす場合

        .box{
            position: relative;
            left:50%;
            transform: translate(-50%);
        }

position:absolute;

position:relativeは親要素に対する相対的な位置を決める

親要素に対しての自分の位置を設定(親にpositionを指定していないと動かない)

何も設定していない場合

        .box{
            background-color: aqua;
            height: 100px;
            width: 200px;
            position: relative;
            top:-50%;
            left:50%;
            transform: translate(-50%);
        }
        .box2{
            background-color:blueviolet;
            height: 50px;
            width: 100px;
            position: absolute;
        }

真ん中に設定する場合(.boxの設定は同じ)

            .box3{
                background-color:blueviolet;
                height: 50px;
                width: 100px;
                position: absolute;
                top:50%;
                left:50%;
                transform: translate(-50%,-50%);
            }

display:flex;

よくわからないけど使うと子要素が横並びになるしなんか打ち込めば真ん中になる

真ん中にする場合

                .box4{
                    display: flex;
                    background-color: aqua;
                    height: 100px;
                    width: 200px;
                    justify-content: center;
                    align-items: center;    
                    position: relative;
                    top:-50%;
                    left:50%;
                    transform: translate(-50%);
                }
                .box5{
                    background-color:blueviolet;
                    height: 50px;
                    width: 80px;
                }

使うと子要素が横並びになる

                    .box4{
                        display: flex;
                        background-color: aqua;
                        height: 100px;
                        width: 200px;
                        justify-content: center;
                        align-items: center;
